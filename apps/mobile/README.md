## Getting Started

### Prerequisites
- Flutter SDK (>=3.2.0)
- Xcode (for iOS)
- Android Studio (for Android)

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Set up environment variables (copy `.env.example` to `.env`)
4. Run the app:
   ```bash
   flutter run
   ```
